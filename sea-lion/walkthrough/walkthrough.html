LiTeCAB

Life Cycle Assessment & Techno-Economic Analysis
for Battery Processes
A desktop tool for evaluating environmental impacts, costs, and material circularity of lithium-ion battery manufacturing and recycling processes. This walkthrough reflects the current startup, inventory, and analysis workflow.
Step 1
Welcome Screen
Launch LiTeCAB to open the startup screen. Choose Load Saved to open an existing .json project, or New Project to start from templates or uploaded data. Use the Walkthrough button at bottom-right anytime for this guide.
Step 2
Select a Template
Choose from built-in manufacturing and recycling templates. Scroll through categories, then use Download Blank Template or Upload Process Template for custom workflows. Built-in templates include pre-computed TRACI v2.1 scores and cost fields.
Step 3
Create Project
Enter a project name and confirm chemistry/category when needed. Click Create Project to open the model. If a project name already exists, LiTeCAB prompts before overwrite.
Step 4
Workspace Overview
The main workspace has a left process view and a right panel with Inventory and Analysis tabs. Select any node to inspect parameters, outputs, and inputs. Use the status bar to track project name, process count, chemistry, and save/analysis messages.
Step 5
Select a Process to Edit
Click a process node in the diagram to focus its inventory. The right panel switches to process-specific inputs, and the bottom specs panel shows duration, temperature, equipment, and source information.
Step 6
Add Input from Ecoinvent Lookup
Use + Add Input, search the lookup dataset, select an activity, then set quantity, unit, and cost before adding. The selected item includes attached TRACI data for LCA calculations.
Step 7
Add Cost-Only Items
Use + Cost Only to add non-LCA items such as equipment, labor, and transport. These entries contribute to TEA while leaving LCA impacts unchanged.
Step 8
Run LCA
In the Analysis tab, run Life Cycle Assessment (LCA) to compute TRACI v2.1 impacts. Expand categories to review process-level contribution breakdowns.
Step 9
Run TEA
Run Techno-Economic Analysis (TEA) for total and per-process cost breakdowns. Edits in Inventory flow directly into TEA. Use the menu save action to Save As your current project state.
Step 10
Circularity Metrics
Run Circularity to evaluate material circularity across key battery materials. Then export consolidated results to Excel for reporting and comparison across scenarios.

Ready to Get Started?

LiTeCAB Desktop is a standalone application — no internet, database, or Python installation required. Download, install, and start analyzing.
